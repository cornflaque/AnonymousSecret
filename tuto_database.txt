Télécharger mongodb : https://www.mongodb.com/download-center/community
Créer un dossier data dans le projet
Dans un invité de commande se placer dans le dossier bin de mongodb
Exemple : C:\Program Files\MongoDB\Server\4.0\bin
Lancer dans l'invité de commande : mongod --dbpath 'chemin vers data'
Dans un second terminal aller dans le dossier bin de mongodb
Et lancer : mongo
Ensuite : use database
Copier ensuite le contenu de 'database.txt' dans l'invité de commande et exécuter
Lancer ensuite : db.usercollection.insert(donnees);
La base doit alors être remplie, pour le vérifier lancer : db.usercollection.find().pretty()
Le contenu de la base doit alors s'afficher
